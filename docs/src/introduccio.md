
# Introducció

Aquest llibre documenta l’activitat **"Sass i CSS modular"** realitzada amb GitHub Classroom.

## Objectius
- Modularitzar estils amb **Sass/SCSS**.
- Reutilitzar **variables, mixins i funcions**.
- Implementar **components** coherents i accessibles.
- Configurar el **build** i publicar la documentació amb **mdBook**.

## Estructura del projecte (resum)
```
src/
  scss/
    _variables.scss
    _mixins.scss
    _functions.scss
    _base.scss
    _layout.scss
    _components.scss
    _utilities.scss
    main.scss
  index.html

dist/css/
  main.css
  main.min.css

docs/ (mdBook)
  book.toml
  src/
    SUMMARY.md
    ...
```
